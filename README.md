## Overview

### Tech Stack

- Next.js 15
- Tailwind CSS v4
- shadcn/ui

### Featured

- Clean & modern design
- Light & Dark theme support
- SEO optimization: [JSON-LD schema](https://json-ld.org), sitemap, robots
- AI-friendly [/llms.txt](https://llmstxt.org)
- Spam-protected email and phone number
- Installable PWA
